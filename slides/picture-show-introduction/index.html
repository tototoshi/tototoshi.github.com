Why Picture Show ?

Slideshows should be relatively portable and should not require proprietary formats to run. Slideshow presentations should be as simple as possible for an audience to understand. Software should be the same way.

インストール

必要なもの

  • sbt
  • conscript

手順1

 $ git clone git://github.com/softprops/picture-show.git
 $ cd picture-show
 $ sbt publish-local

手順2

 $ cs softprops/picture-show

以上！
