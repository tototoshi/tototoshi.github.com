# インストール

!SLIDE

## 必要なもの

* [sbt](https://github.com/harrah/xsbt "harrah/xsbt - GitHub")
* [conscript](https://github.com/n8han/conscript "n8han/conscript - GitHub")

!SLIDE

## 手順1
```sh
 $ git clone git://github.com/softprops/picture-show.git
 $ cd picture-show
 $ sbt publish-local
```

!SLIDE

## 手順2
```sh
 $ cs softprops/picture-show
```
