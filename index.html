Libraries & Tools

The followings are libraries and tools I have written. Almost all are implemented in Scala. It seems that Scala is now my primary language.

  • play-ascii-art-plugin shows ascii-art on starting/stopping Play apps.
  • play-json4s is a library to integrate json4s to Play2.0.
  • lift-json-play-module is a library to integrate lift-json to Play2.0.
  • simple-slides is a simple presentation tool.
  • helm-find-files-in-project is a emacs plugin to jump to files in current project
  • scala-csv is a simple opencsv wrapper.
  • scala-http-client is a http client library for Scala. It wraps apache httpcomponent.
  • sawk is my experiment to write scala like awk
  • mvnsearch is a command line tool to search Java/Scala libraries in maven repository.
  • scala-base64 is a Base64 encoder for scala
  • hatenacala is a command line tool for hatena diary. It also works well with emacs.

See my github in more detail.
